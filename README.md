# JavaScript Math Library

This is an exercise for unit testing assignment, 
that contains simple JavaScript library for basic arithmetic operations

## Features
* **Arithmetic Operations:** Addition, subtraction, multiplication, and division.
* **Error Handling:** Throws an error when not using numbers or trying to divide by zero.
* **Unit Tests:** Testing is done with **Mocha** and **Chai**.

## Getting Started

### Prerequisites
* Node.js installed.

### Installation
1.  Clone the repository.
2.  Install dependencies (Mocha & Chai):
```bash
    npm install
```

## Usage

To run the main program example:
```bash 
    node index.js
```

## Testing

To run the automated unit tests:
```bash
    npm test
```
